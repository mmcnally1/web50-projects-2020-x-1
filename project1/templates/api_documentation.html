API Documentation

If you make a request to /api/isbn, where isbn is the book's ISBN, you will receive a JSON object containing the book's ISBN, title, author, year published, average score (1-5 scale), and number of ratings in the format of:

{"isbn": "0553803700",

"title": "I, Robot",

"author": "Isaac Asimov",

"year": "1950",

"rating": "4.8",

"num_reviews": "15"}
